Archi Plugin Packager Notes
---------------------------

1. Select the project "org.archicontribs.modelrepository"
2. From the context menu, select "Export..." and then "Deployable plug-ins and fragments"
3. Click "Next" and ensure the project is selected and the output dir is "C:\" (if this is different, edit the packager.xml output.dir location)
4. Finish.
5. Run the packager.xml Ant script

The resulting zip file is the packaged plug-in.
